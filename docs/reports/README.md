# Reports

Static HTML reports about the slime repository:

- [`slime_architecture.html`](./slime_architecture.html) — Repository structure and module interaction with code snippets.
- [`harbor_integration.html`](./harbor_integration.html) — Changes needed to support agentic RL training on tool-based
  environments specified in the [Harbor](https://github.com/harbor-framework/harbor) framework format.

Open the files directly in a browser; they are self-contained (single-file HTML + inline CSS, no external assets).
